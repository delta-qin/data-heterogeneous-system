企业数据异构系统  SaaS平台


本平台集异构数据集成、数据分析、智能预警等功能于一体，提供一套完善、强大、便捷的综合数据处理工具

通过挖掘大数据隐藏信息，为决策者提供决策依据，有效提高企业办公效率及准确性。


在这里，你可以：

  • 打破信息壁垒，实现互联互通
  • 快速集成数据，实现统一管理
  • 结合数据挖掘，实现信息利用

功能介绍

数据集成--异构数据源

有机集中不同来源、格式、性质特征的数据，为企业提供全面的数据共享。

联通信息孤岛

跨越多部门信息的封闭壁垒，整合统一，集中处理，满足企业日益剧增的内外部信息交互需求

解决数据异构

中间件模式提供统一模式和接口，集成数据源在管理系统、通信协议、数据模式、数据类型、取值及语义上的异构性

优化分布传输

多方机制加强网络传输性能，维护信息传输安全，解决数据源异地分布传输问题

数据可视化--多种图表拖拽

用户友好图形化界面，高效传达数据信息，美学与功能齐头并进

页面大屏展示

提供大屏数据展示，面板灵活配置，满足用户多样化业务需求

丰富的呈现形式

数据引擎开发采取多种呈现形式，完美应对大数据时代读取需求

优越图形化能力

集成数据化采用前沿技术，超高画面质感，给用户愉悦使用体验

集群监控--机器学习+神经网络预警

系统监控成千上万日夜不息的服务器，稳定支撑企业运转的核心力量

智能预测报警

深度学习训练预测系统，在故障发生前进行预警，将事故代价最小化

日志管理分析

采用ELK日志处理系统，根据关键信息准确定位服务器，及时发现问题，解决问题

节省服务器开销

系统通过Redis储存高频使用数据，极大提升速度，节约内存开销，数据持久化保证安全

产品架构

技术要点

异构数据库

支持influxDB,MySQL, SQLServer,MongoDB,Postgresql等多种数据库数据源，满足用户的多方异构数据源需求。

智能算法

平台集成机器学习算法Arima，以及深度学习算法RNN，LSTM，GRU对机器未来的负载做出预测，即使通知用户。

微服务架构

使用最流行的spring cloud alibaba架构体系，实现模块解耦，加速系统开发，减轻系统发布维护压力。

分库分表

针对大量数据的需要实现分库分表，提高数据处理效率。分布式ID雪花算法保证不会出现重复ID，保证数据库的一致性

分布式presto

经典三节点架构，一个coordinator负责调度，两个worker负责计算，解决系统查询瓶颈，获取更加高效的查询

稳定可靠

平台使用sentnel作为服务限流降级熔断的工具，网络架构稳定，故障紧急处理机制完善，保证数据快速迁移和恢复。

全面保护

平台对用户账号及个人信息使用 SpringSecurity + OAuth2 +JWT 进行非对称加密保护，保障用户业务服务稳定进行。

可视面板

拥有功能齐全的度量仪表盘和图形编辑器，支持influxDB、MySQL等多种数据源，图形风格多样，页面布局美观
